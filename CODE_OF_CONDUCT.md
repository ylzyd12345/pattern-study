# 行为准则

## 我们的承诺

为了营造一个开放和欢迎的环境，作为贡献者和维护者，我们承诺参与我们项目和社区的每个人都能享有无骚扰的体验，无论年龄、体型、残疾、种族、性别认同和表达、经验水平、国籍、个人外貌、种族、宗教或性取向如何。

## 我们的标准

有助于创造积极环境的行为示例包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为示例包括：

- 使用性化的语言或图像以及不恰当的性关注或进展
- 挑衅、侮辱/贬低性评论以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，发布他人的私人信息，如物理或电子地址
- 其他在专业环境中被合理认为不适当的行为

## 我们的责任

项目维护者有责任明确可接受行为的标准，并对任何不可接受行为的实例采取适当和公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki编辑、问题和其他贡献，或暂时或永久禁止任何贡献者从事其他维护者认为不适当、威胁、冒犯或有害的行为。

## 范围

本行为准则适用于项目空间内的所有行为，也适用于个人在公共空间代表项目或其社区时的行为。代表项目或社区的示例包括使用官方项目电子邮件地址、通过官方社交媒体账号发布信息，或作为指定代表参加在线或离线活动。

## 执法

可通过联系项目团队（your-email@example.com）举报滥用、骚扰或其他不可接受行为的实例。所有投诉都将被及时和公平地审查和调查。

所有项目维护者都有义务尊重任何事件报告者的隐私和安全。

## 执法指南

项目维护者将遵循这些社区影响指南来确定对任何他们认为违反本行为准则的行为的后果：

### 1. 纠正

**社区影响**：使用不专业或不欢迎的语言，或其他被确定为不符合专业行为的行为。

**后果**：由项目维护者对违规者进行私人的书面警告，明确说明违规行为的性质，并解释其违反了行为准则。可能需要公开道歉。

### 2. 警告

**社区影响**：在单个事件或一系列行动中违反行为准则。

**后果**：警告并附带具体行为改变要求。在指定时间段内，可能禁止与相关人员互动，包括主动与执行团队联系。违反这些条款可能导致临时或永久禁止。

### 3. 临时禁止

**社区影响**：严重违反行为准则，包括持续的不当行为。

**后果**：在指定时间段内，临时禁止所有公共互动和参与。在这段时间内，不得与相关人员或团体进行公开或私下的互动，包括主动与执行团队联系。违反这些条款可能导致永久禁止。

### 4. 永久禁止

**社区影响**：表现出对行为准则的严重侵犯，包括持续的不当行为、骚扰个人或攻击或贬低类别的个人。

**后果**：永久禁止所有公共互动和参与。
